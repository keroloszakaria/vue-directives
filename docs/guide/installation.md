# Installation

## Package Manager

::: code-group

```bash [npm]
npm install vue-directives
```

```bash [yarn]
yarn add vue-directives
```

```bash [pnpm]
pnpm add vue-directives
```

:::

## Requirements

- **Vue 3.2+** (peer dependency)
- **Nuxt 3** (optional, for Nuxt module usage)
- **TypeScript 4.7+** (optional, for type support)
